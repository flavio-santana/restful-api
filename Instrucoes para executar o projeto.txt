Configurando o Ambiente 

Obs: O projeto foi elaborado utilizando os seguintes recursos:  
    
    - Ubuntu versão 16.04 LTS
    - PHP 7.2.30-1
    - Laravel Framework 7.6.2
    - Apache/2.4.18
    - MySQL 5.7.29-0ubuntu0.16.04.1

Pré-requisitos 

    - Instalar o gerenciador de pacotes Composer

Passo a passo para executar o projeto 

    1.  Execute o código abaixo para criar o projeto 
        
        composer create-project --prefer-dist laravel/laravel rentmyroom 

    2.  Para criar as tabelas na base de dados execute o comando, a seguir: 

        php artisan migrate  

    3.  Para popular as tabelas do banco de dados, execute a instrução abaixo: 

        php artisan db:seed
